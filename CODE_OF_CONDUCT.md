# Code of Conduct

## Our Standard

We want this project to be useful, direct, and respectful. Contributors are expected to:

- communicate clearly and in good faith
- assume mistakes before malice
- accept technical feedback without personal attacks
- avoid harassment, discrimination, and disruptive behavior

## Unacceptable Behavior

- personal attacks, insults, or intimidation
- discriminatory language or conduct
- deliberate harassment, dogpiling, or stalking
- publishing private information without permission
- repeated bad-faith interaction that blocks collaboration

## Enforcement

Project maintainers may remove comments, reject contributions, or restrict participation for behavior that harms the project or its contributors.

## Scope

This applies in repository discussions, issues, pull requests, and other project communication channels.
